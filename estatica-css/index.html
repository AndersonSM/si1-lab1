Sistemas de Informação 1

Bem vindo à página da disciplina Sistemas de Informação 1. Aqui você encontrará informações sobre esta discplina que podem ser relevantes para você.

Por que estudar SI?

Considerando que nesta disciplina são abordados conceitos relacionados à design de software, todo programador deve saber a diferença entre simplesmente programar e fazer um código de boa qualidade, para que seus projetos possam sempre ser expandidos e alterados sem grandes problemas. Para isso, é necessário aprender e discutir técnicas que facilitam essa boa prática. Além disso, o foco da disciplina é em WEB. Portanto, você terá a oportunidade de aprender novas ferramentas e tecnologias, além de desenvolver sua habilidade de fazer isso por conta própria.

O que aprenderei?

No final do curso, você deverá ser capaz de:

  1. Descrever os objetivos do projeto de software e seu papel no desenvolvimento de um software
  2. Definir o que são padrões de projeto e nomear um conjunto de padrões
  3. Criticar a necessidade e/ou oportunidade da aplicação de padrões em um contexto
  4. Descrever e criticar uma arquitetura de software
  5. Aplicar boas práticas e padrões de projeto na transformação de uma especificação em um projeto flexível e reutilizável
  6. Produzir um software complexo utilizando boas práticas e padrões de projeto
  7. Produzir um software web que utilize tecnologias atualmente populares no mercado
  8. Conectar o software sendo implementado às necessidades do cliente que o demanda
  9. Criar software multithreaded
